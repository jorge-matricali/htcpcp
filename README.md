# htcpcp

An implementation of Hyper Text Coffee Pot Control Protocol (HTCPCP/1.0)
defined in [RFC-2324](https://tools.ietf.org/html/rfc2324).

```
usage: ./htcpcpd [-h] [-v] [-p <port>]

  -h                This help
  -v                Verbose mode
  -p <port>         Port number
```
